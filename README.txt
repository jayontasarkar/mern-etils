# ETILS - Doc sharing app

## Backend setup
### All backend codes are included into backend folder. Just duplicate the .env.example file or rename it to .env file, fill the environment variables, generate a uploads folder in root of backend directory and follow the following steps

``run npm install or yarn``

``run yarn dev or npm run dev``

## Frontend setup
### All frontend codes are included into frontend folder. Just duplicate the .env.example file or rename it to .env file, fill the environment variables and follow the following steps

``run npm install or yarn``

``run yarn start or npm run start``
